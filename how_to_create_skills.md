# 🎯 스킬 만드는 법 — Hunter World 플러그인 가이드

> 플러그인 UI의 **스킬 DB** 탭에서 스킬을 직접 추가/수정/삭제할 수 있습니다.
> 이 문서는 스킬을 설계할 때 필요한 **모든 필드**와 **밸런스 기준치**를 설명합니다.

---

## 1. 스킬 추가 방법

1. 플러그인 UI에서 **DB** 탭 클릭
2. **스킬** 서브탭 클릭
3. **"새 스킬"** 버튼 클릭
4. 아래 필드들을 채운 후 **"저장"** 클릭

> 💡 **내장 스킬 수정**: 내장 스킬 목록에서 원하는 스킬을 클릭하면 편집기에 로드됩니다.  
> 저장하면 커스텀 오버라이드로 저장되고, "원본 복원"으로 되돌릴 수 있습니다.

---

## 2. 스킬 필드 설명

| 필드 | 설명 | 예시 |
|------|------|------|
| **ID** | 영어 camelCase, 고유값 | `fireBall`, `healingWind` |
| **이름** | 한글 스킬명 | `파이어볼`, `치유의 바람` |
| **랭크** | E / D / C / B / A / S | `C` |
| **카테고리** | 스킬 종류 (아래 표 참고) | `singleAttack` |
| **대상** | 스킬 대상 | `singleEnemy` |
| **계수** | 피해/회복 배율 (아래 기준표 참고) | `3.0` |
| **MP 비용** | 마나 소모량 | `40` |
| **SP 비용** | 스태미나 소모량 | `20` |
| **피해 타입** | `physical` 또는 `magic` | `magic` |
| **속성** | 원소 속성 | `fire` |
| **스탯 타입** | 데미지 계산에 쓰이는 스탯 (쉼표 구분) | `int`, `str, agi` |
| **지속 턴** | 버프/디버프 지속 시간 | `3` |
| **CC 종류** | 상태이상 종류 | `stun` |
| **CC 턴** | 상태이상 지속 턴수 | `1` |
| **버프 스탯** | 올려줄 스탯 이름 | `agi` |
| **버프 수치** | 올려줄 스탯 양 | `4` |
| **상태이상** | 부여할 상태이상 | `poison` |
| **상태이상 턴** | 상태이상 지속 턴 | `3` |
| **설명** | 스킬 설명 텍스트 | `화염 속성 단일 마법 공격.` |

---

## 3. 카테고리 종류

| 카테고리 | 코드 | 대상 | 설명 |
|----------|------|------|------|
| 단일공격 | `singleAttack` | `singleEnemy` | 적 1체에게 피해 |
| 광역공격 | `aoeAttack` | `allEnemies` | 전체 적에게 피해 |
| 단일CC | `singleCC` | `singleEnemy` | 적 1체 피해 + 상태이상 |
| 광역CC | `aoeCC` | `allEnemies` | 전체 적 피해 + 상태이상 |
| 단일힐 | `singleHeal` | `singleAlly` | 아군 1체 회복 |
| 광역힐 | `aoeHeal` | `allAllies` | 전체 아군 회복 |
| 버프 | `buff` | `allAllies` / `self` | 스탯 증가 |
| 유틸리티 | `utility` | 다양 | 자원 회복 등 |

## 5. 상태이상 종류

| 코드 | 한글 | 효과 |
|------|------|------|
| `stun` | 기절 | 행동 불가 |
| `bind` | 속박 | 행동 불가 |
| `sleep` | 수면 | 행동 불가 (피격 시 해제) |
| `poison` | 독 | 매턴 DoT |
| `bleed` | 출혈 | 매턴 DoT + 힐 50% 감소 |
| `burn` | 화상 | 매턴 DoT |
| `curse` | 저주 | 회복량 50% 감소 |
| `silence` | 침묵 | 마법 스킬 사용 불가 |
| `slow` | 둔화 | 명중 -30% / 회피 -50% |

## 6. ⚖️ 밸런스 기준치 (가장 중요!)

### 단일 공격 계수 기준

| 등급 | 계수 범위 | 비용 범위 |
|:---:|:---:|:---:|
| **E** | 1.2 ~ 1.5 | 20 ~ 40 |
| **D** | 1.92 ~ 2.4 | 25 ~ 45 |
| **C** | 2.88 ~ 3.6 | 30 ~ 50 |
| **B** | 4.8 ~ 6.0 | 40 ~ 60 |
| **A** | 7.68 ~ 9.6 | 55 ~ 80 |
| **S** | 11.52 ~ 14.4 | 70 ~ 100 |

### 파생 규칙

| 유형 | 공식 |
|------|------|
| **광역공격** | 단일 계수 × **0.58** |
| **단일CC** | 광역 계수 수준 + CC 판정 추가 |
| **광역CC** | 단일CC 계수 × **0.5**, 비용 × **2** |
| **버프** | 비용만 사용 (계수 불필요) |

### 버프 스탯 보너스 기준

| 등급 | 스탯 증가량 |
|:---:|:---:|
| E | +2 |
| D | +4 |
| C | +6 |
| B | +8 |
| A | +10 |
| S | +12 ~ +15 |

## 7. 데미지 공식

```
기본값 = (2 × 주스탯) + (3 × ATK)
최종 피해 = 기본값 × 계수 × 크리배율 × 속성보정 × 저항보정 × 방어계산
```

### 회복 공식

```
회복량 = (2 × INT + 3 × ATK) × 계수 × 저주보정
출혈 대상 → 회복량 50% 감소
```

## 9. 자주 하는 실수

| 실수 | 해결 |
|------|------|
| ID에 한글/공백 사용 | 영어 camelCase만 사용 (`mySkill`) |
| 광역 스킬에 단일 계수 넣기 | 광역은 단일 × 0.58 이하 |
| 버프에 계수 넣기 | 버프는 계수 불필요, 버프 스탯/수치 설정 |
| S급 계수를 E급에 넣기 | 등급별 기준표 확인 |
| CC 종류 없이 singleCC 설정 | CC 종류 필수 입력 |

---

## 10. 전체삭제

**"스킬 전체삭제"** 버튼을 누르면 모든 커스텀 스킬이 삭제됩니다.  
내장 스킬(BUILTIN_SKILLS 153종)은 삭제되지 않습니다.
