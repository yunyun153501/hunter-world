# 🎯 스킬 만드는 법 — Hunter World 플러그인 가이드

> 플러그인 UI의 **스킬 DB** 탭에서 스킬을 직접 추가/수정/삭제할 수 있습니다.
> JSON 가져오기로 여러 스킬을 한번에 추가할 수도 있습니다.
> 이 문서는 스킬을 설계할 때 필요한 **모든 필드**와 **밸런스 기준치**를 설명합니다.

---

## 1. 스킬 추가 방법

### 방법 1: 수동 추가
1. 플러그인 UI에서 **DB** 탭 클릭
2. **스킬** 서브탭 클릭
3. **"새 스킬"** 버튼 클릭
4. 아래 필드들을 채운 후 **"저장"** 클릭

### 방법 2: JSON 가져오기
1. 스킬 편집기 아래의 **스킬 JSON 가져오기** 영역에 JSON 붙여넣기
2. **"JSON 가져오기"** 클릭
3. 배열 `[...]` 또는 `{"skills":[...]}` 형식 모두 지원
4. 같은 ID는 덮어쓰기, 여러 스킬 한번에 추가 가능

> 💡 처음 시작하면 **⭐ 스킬가이드** 샘플이 들어있습니다. 참고 후 삭제하세요.

## 2. 스킬 필드 설명

| 필드 | 설명 | 예시 |
|------|------|------|
| **ID** | 영어 camelCase, 고유값 | `fireBall`, `healingWind` |
| **이름** | 한글 스킬명 | `파이어볼`, `치유의 바람` |
| **랭크** | E / D / C / B / A / S | `C` |
| **카테고리** | 스킬 종류 (아래 표 참고) | `singleAttack` |
| **대상** | 스킬 대상 (아래 대상 표 참고) | `singleEnemy` |
| **계수** | 피해/회복 배율 (**대상+등급 선택 시 하한값 자동 입력**, 수정 가능) | `2.88` |
| **MP 비용** | 마나 소모량 | `40` |
| **SP 비용** | 스태미나 소모량 | `20` |
| **피해 타입** | `physical` 또는 `magic` | `magic` |
| **속성** | 원소 속성 | `fire` |
| **스탯 타입** | 데미지 계산에 쓰이는 스탯 (쉼표 구분) | `int`, `str, agi` |
| **지속 턴** | 버프/디버프 지속 시간 | `3` |
| **CC 종류** | 드롭다운: stun/bind/sleep/silence/slow | `stun` |
| **CC 턴** | 상태이상 지속 턴수 | `1` |
| **CC 확률(0~1)** | CC 적용 확률 (비우면 100%) | `0.3` → 30% |
| **버프 스탯** | 드롭다운: str/con/int/agi/sense | `agi` |
| **버프 수치** | 올려줄 스탯 양 | `4` |
| **상태이상** | 드롭다운: poison/bleed/burn/curse/silence/slow | `poison` |
| **상태이상 턴** | 상태이상 지속 턴 | `3` |
| **상태이상 확률(0~1)** | 상태이상 적용 확률 (비우면 기본값) | `0.28` → 28% |
| **쿨타임(턴)** | 사용 후 재사용 불가 턴 수 (0=없음) | `2` |
| **설명** | 스킬 설명 텍스트 | `화염 속성 단일 마법 공격.` |

## 3. 카테고리 종류

| 카테고리 | 코드 | 대상 | 설명 |
|----------|------|------|------|
| 단일공격 | `singleAttack` | `singleEnemy` | 적 1체에게 피해 |
| 광역공격 | `aoeAttack` | `allEnemies` | 전체 적에게 피해 |
| 단일CC | `singleCC` | `singleEnemy` | 적 1체 피해 + 상태이상 |
| 광역CC | `aoeCC` | `allEnemies` | 전체 적 피해 + 상태이상 |
| 단일힐 | `singleHeal` | `singleAlly` | 아군 1체 회복 |
| 광역힐 | `aoeHeal` | `allAllies` | 전체 아군 회복 |
| 버프 | `buff` | `allAllies` / `self` | 스탯 증가 |
| 유틸리티 | `utility` | 다양 | 자원 회복 등 |

## 3-1. 대상(Target) 종류

| 대상 | 코드 | 설명 | 계수 배율 |
|------|------|------|:---------:|
| 단일 적 | `singleEnemy` | 적 1체 | ×1.0 |
| 전체 적 | `allEnemies` | 모든 적 (광역) | ×0.58 |
| 전열 적 | `rowFront` | 전열(front) 적만 (광역) | ×0.58 |
| 중열 적 | `rowMid` | 중열(mid) 적만 (광역) | ×0.58 |
| 후열 적 | `rowBack` | 후열(back) 적만 (광역) | ×0.58 |
| 전열+중열 | `rowFrontMid` | 전열+중열 적 (2열 공격) | ×0.58 |
| 중열+후열 | `rowMidBack` | 중열+후열 적 (2열 공격) | ×0.58 |
| 단일 아군 | `singleAlly` | 아군 1체 | ×1.0 |
| 전체 아군 | `allAllies` | 모든 아군 | ×0.58 |
| 자신 | `self` | 시전자 자신 | ×1.0 |

> 💡 **자동 계수 입력**: 대상 또는 등급을 선택하면 계수가 자동으로 입력됩니다. 직접 수정도 가능합니다.
> 💡 **열 공격 폴백**: 해당 열에 생존한 적이 없으면 **가장 앞 열의 생존자**를 공격합니다.

## 5. 상태이상 종류 (드롭다운 선택)

### CC 종류 (행동 방해)

| 코드 | 한글 | 효과 |
|------|------|------|
| `stun` | 기절 | 행동 불가 (이후 5턴 면역) |
| `bind` | 속박 | 행동 불가 |
| `sleep` | 수면 | 행동 불가 (피격 시 해제, 이후 5턴 면역) |
| `silence` | 침묵 | 마법 스킬 사용 불가 |
| `slow` | 둔화 | 명중 -30% / 회피 -50% |

### 상태이상 (DoT/디버프)

| 코드 | 한글 | 효과 | 기본 확률 | 기본 턴 |
|------|------|------|:---------:|:-------:|
| `poison` | 독 | 매턴 DoT (최대 3중첩) | 28% | 3턴 |
| `bleed` | 출혈 | 피격 시 30% 즉시 추가피해 + 힐 50% 감소 | 28% | 3턴 |
| `burn` | 화상 | 매턴 DoT (최대 5중첩) + **받는 데미지 +10%** | 28% | 5턴 |
| `curse` | 저주 | 회복량 감소 (등급별 E:10%~S:30%) | 24% | 3턴 |
| `silence` | 침묵 | 마법 스킬 사용 불가 | 20% | 3턴 |
| `slow` | 둔화 | 명중 -30% / 회피 -50% | 25% | 3턴 |

> 💡 상태이상 확률을 비워두면 위의 기본 확률이 적용됩니다.

### 상태이상 데미지 공식

```
기본값(rawBase) = (2 × 주스탯) + (3 × ATK)

독(poison):  매턴 기본값 × 계수 × 0.2 × 중첩수 (최대 3중첩)
화상(burn):  매턴 기본값 × 계수 × 0.12 × 중첩수 (최대 5중첩)
             + 받는 데미지 +10% (중첩 무관, 화상 걸린 동안 지속)
출혈(bleed): 피격 시 해당 피해의 30% 즉시 추가피해 (DoT 아님)
             + 3턴간 회복량 50% 감소
저주(curse): 등급별 공격력/회복량 감소 (E:10% ~ S:30%)
```

### E급 순수공격 예시 (주스탯15, ATK5, 계수1.35)

```
기본값 = (2×15) + (3×5) = 45
상한 직접피해 = 45 × 1.5 = 67.5
스킬 데미지 = 45 × 1.35 = 60.75
```

### E급 CC/상태이상 예시 (추천계수 0.96)

```
직접피해 = 45 × 0.96 = 43.20

독 1중첩 3턴합: 45 × 0.96 × 0.2 × 3 = 25.92
  → 총합: 43.20 + 25.92 = 69.12 (상한의 102%) ✓
화상 1중첩 5턴합: 45 × 0.96 × 0.12 × 5 = 25.92
  → 총합: 43.20 + 25.92 = 69.12 + 피격+10% ✓
출혈 즉시추가: 43.20 × 0.3 = 12.96
  → 총합: 43.20 + 12.96 = 56.16 (상한83%) + 회복반감 ✓
```

### 버프 스탯 (드롭다운 선택)

| 코드 | 한글 |
|------|------|
| `str` | 힘 |
| `con` | 체질 |
| `int` | 지능 |
| `agi` | 민첩 |
| `sense` | 감각 |

## 6. ⚖️ 밸런스 기준치 (가장 중요!)

### 단일 공격 계수 기준 (자동 입력 = 하한값)

| 등급 | 계수 범위 | 자동 입력 (하한값) | 비용 범위 |
|:---:|:---:|:---:|:---:|
| **E** | 1.2 ~ 1.5 | **1.2** | 20 ~ 40 |
| **D** | 1.92 ~ 2.4 | **1.92** | 25 ~ 45 |
| **C** | 2.88 ~ 3.6 | **2.88** | 30 ~ 50 |
| **B** | 4.8 ~ 6.0 | **4.8** | 40 ~ 60 |
| **A** | 7.68 ~ 9.6 | **7.68** | 55 ~ 80 |
| **S** | 11.52 ~ 14.4 | **11.52** | 70 ~ 100 |

### 파생 규칙

| 유형 | 공식 | 자동 적용 |
|------|------|-----------|
| **광역공격** | 단일 계수 × **0.58** | ✅ 대상 선택 시 자동 |
| **열 공격(row)** | 단일 계수 × **0.58** | ✅ 대상 선택 시 자동 |
| **단일CC/상태이상** | 순수 하한 × **0.8** (20% 약화) | 수동 (추천) |
| **광역CC/상태이상** | 단일CC 추천 × **0.58** | 수동 (추천) |
| **광역CC** | 단일CC 계수 × **0.5**, 비용 × **2** | — |
| **버프** | 비용만 사용 (계수 불필요) | — |

### CC/상태이상 스킬 추천 계수 (하한 × 0.8)

> 상태이상이 붙는 스킬은 직접 데미지를 낮추는 대신, 상태이상(DoT/디버프) 효과로 총 가치를 보상.
> **직접피해 + DoT 총합 ≥ 순수공격 상한계수급** 이면 적절한 밸런스.

| 등급 | 단일 CC/상태이상 | 광역 CC/상태이상 |
|:---:|:---:|:---:|
| **E** | 0.96 | 0.56 |
| **D** | 1.54 | 0.89 |
| **C** | 2.30 | 1.33 |
| **B** | 3.84 | 2.23 |
| **A** | 6.14 | 3.56 |
| **S** | 9.22 | 5.35 |

> 독/화상: 직접+DoT 총합 ≈ 상한의 102%  
> 출혈: 직접+즉시추가 ≈ 상한의 83% + 회복반감 유틸리티  
> 상태이상이 강할수록 계수를 더 낮춰도 됨

### 버프 스탯 보너스 기준

| 등급 | 스탯 증가량 |
|:---:|:---:|
| E | +2 |
| D | +4 |
| C | +6 |
| B | +8 |
| A | +10 |
| S | +12 ~ +15 |

## 7. 데미지 공식

```
기본값 = (2 × 주스탯) + (3 × ATK)
최종 피해 = 기본값 × 계수 × 크리배율 × 속성보정 × 저항보정 × 방어계산
```

### 회복 공식

```
회복량 = (2 × INT + 3 × ATK) × 계수 × 저주보정
출혈 대상 → 회복량 50% 감소
```

## 8. JSON 가져오기 예시

```json
[
  {
    "id": "fireBall",
    "name": "파이어볼",
    "grade": "C",
    "category": "singleAttack",
    "target": "singleEnemy",
    "costs": { "mp": 35, "sp": 0 },
    "coef": 3.24,
    "statTypes": ["int"],
    "damageType": "magic",
    "element": "fire",
    "cooldown": 2,
    "desc": "C급 화염 단일 마법 공격. 쿨타임 2턴."
  },
  {
    "id": "frontSlash",
    "name": "전열 참격",
    "grade": "D",
    "category": "singleAttack",
    "target": "rowFront",
    "costs": { "mp": 0, "sp": 30 },
    "coef": 1.62,
    "statTypes": ["str"],
    "damageType": "physical",
    "element": "none",
    "desc": "전열 적 전체를 베는 반광역 공격."
  }
]
```

> 💡 `cooldown` 필드가 없거나 0이면 쿨타임 없음. 숫자 입력 시 사용 후 해당 턴 동안 재사용 불가.

## 10. 자주 하는 실수

| 실수 | 해결 |
|------|------|
| ID에 한글/공백 사용 | 영어 camelCase만 사용 (`mySkill`) |
| 광역 스킬에 단일 계수 넣기 | 광역은 단일 × 0.58 이하 |
| 버프에 계수 넣기 | 버프는 계수 불필요, 버프 스탯/수치 설정 |
| S급 계수를 E급에 넣기 | 등급별 기준표 확인 |
| CC 종류 없이 singleCC 설정 | CC 종류 드롭다운에서 선택 |
| 쿨타임 미설정 | 강력한 스킬은 쿨타임 2~3턴 권장 |
