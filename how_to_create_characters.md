# 👤 캐릭터 만드는 법 — Hunter World 플러그인 가이드

> 플러그인 UI의 **DB** 탭 → **캐릭터** 서브탭에서 캐릭터를 추가/수정/삭제할 수 있습니다.
> 이 문서는 캐릭터를 설계할 때 필요한 **모든 필드**와 **스탯 기준치**를 설명합니다.

---

## 1. 캐릭터 추가 방법

1. 플러그인 UI에서 **DB** 탭 클릭
2. **캐릭터** 서브탭 클릭
3. **"새 캐릭터"** 버튼 클릭
4. 아래 필드들을 채운 후 **"저장"** 클릭

> 💡 처음 시작하면 **⭐ 캐릭터 가이드** 샘플이 들어있습니다. 참고 후 삭제하세요.

---

## 2. 캐릭터 필드 설명

| 필드 | 설명 | 예시 |
|------|------|------|
| **ID** | 영어, 고유값 (자동 생성 가능) | `char_yuna` |
| **이름** | 캐릭터 이름 | `최유나` |
| **직업** | 캐릭터 직업 | `크루세이더` |
| **포지션** | 전투 역할 | `전열탱커` |
| **행** | front / mid / back | `front` |
| **등급** | E / D / C / B / A / S | `E` |
| **HP** | 체력 (0이면 스탯 자동 계산) | `100` |
| **MP** | 마나 (0이면 스탯 자동 계산) | `100` |
| **SP** | 스태미나 (0이면 스탯 자동 계산) | `100` |
| **ATK** | 공격력 (기본 0, 장비·스킬로 증가) | `0` |
| **물리방어** | PDEF (기본 0, 장비·스킬로 증가) | `0` |
| **마법방어** | MDEF (기본 0, 장비·스킬로 증가) | `0` |
| **STR** | 힘 (최소 10) | `12` |
| **CON** | 체질 (최소 10) | `19` |
| **INT** | 지능 (최소 10) | `10` |
| **AGI** | 민첩 (최소 10) | `14` |
| **SENSE** | 감각 (최소 10) | `15` |
| **피해 타입** | physical / magic | `physical` |
| **공격 스탯** | 주력 스탯 | `con` |
| **기본 위협값** | 탱커는 높게, 딜러는 낮게 | `0` |
| **스킬 ID** | 보유 스킬 (쉼표 구분) | `shieldBash, taunt` |
| **메모** | 설명 텍스트 | `전열 탱커` |

## 3. 직업 종류

| 직업 | 포지션 | 행 | 특성 |
|------|--------|:---:|------|
| 크루세이더 | 전열탱커 | front | 탱킹 + 도발 |
| 무투가 | 근거리물리 | front | 근접 딜러 |
| 스트라이커 | 근거리물리 | front | 고화력 근접 |
| 투척가 | 원거리물리 | mid | 투척 딜러 |
| 궁수 | 원거리물리 | back | 원거리 물리 딜러 |
| 마법사 | 원거리마법 | back | 마법 딜러 |
| 클레릭 | 힐러 | back | 회복 + 서포트 |
| 정령술사 | 원거리마법 | back | 소환 + 마법 |
| 연금술사 | 후열서포터 | back | 포션 + 지원 |
| 수리가 | 비전투 | back | 장비 수리 |
| 대장장이 | 비전투 | — | 장비 제작 |
| 무직업 | 다양 | 다양 | 범용 |

## 4. 스탯 공식

> ❗ **스탯 최소값 = 10**. 레벨 1에 모든 스탯이 10일 때 HP = MP = SP = **100**.
> HP/MP/SP를 **0**으로 입력하면 스탯 기반으로 **자동 계산**됩니다.
> ATK/물방/마방은 **기본 0** — 장비와 스킬로만 올릴 수 있습니다.

| 파생 스탯 | 공식 | 모든 스탯=10 일 때 |
|-----------|------|----|
| **HP** | `100 + (CON - 10) × 10 + (STR - 10) × 3` | 100 |
| **MP** | `100 + (INT - 10) × 10 + (SENSE - 10) × 3` | 100 |
| **SP** | `100 + (AGI - 10) × 10 + (SENSE - 10) × 3` | 100 |
| **ATK** | 기본 **0** (장비·스킬로 증가) | 0 |
| **PDEF** | 기본 **0** (장비·스킬로 증가) | 0 |
| **MDEF** | 기본 **0** (장비·스킬로 증가) | 0 |

## 5. ⚖️ 등급별 스탯 기준치

| 구분 | E | D | C | B | A | S |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| **최대 레벨** | 15 | 25 | 40 | 60 | 80 | 120 |
| **주스탯 상한** | 25 | 40 | 60 | 80 | 100 | 150 |
| **스탯합 기준** | ~70 | 70~90 | 90~120 | 120~160 | 160~200 | 200~ |

### 스탯 배분 원칙

- **주스탯** (직업 핵심): 등급 상한의 50~80%
- **보조스탯**: 주스탯의 50~70%
- **약점스탯**: 최소 10 유지
- **스탯 합계**: 등급별 기준 범위 내로

### 직업별 주스탯 가이드

| 직업 | 주스탯 | 보조스탯 |
|------|--------|----------|
| 탱커 (크루세이더) | CON | STR, SENSE |
| 근접 딜러 (무투가/스트라이커) | STR | AGI, CON |
| 원거리 물리 (궁수/투척가) | AGI | SENSE, STR |
| 마법사 | INT | SENSE, AGI |
| 힐러 (클레릭) | INT | SENSE, CON |
| 서포터 (연금술사) | INT, SENSE | CON |

## 6. 행 배치

| 행 | 코드 | 포지션 |
|------|------|--------|
| 전열 | `front` | 탱커, 근접 딜러 |
| 중열 | `mid` | 투척가 |
| 후열 | `back` | 궁수, 마법사, 힐러, 서포터 |

> 💡 포지션이 지정되면 행이 자동으로 추론됩니다:
> - 탱커/근거리 → front
> - 투척 → mid
> - 원거리/궁수/마법사/힐러 → back

---

## 7. 스킬 배정

- **스킬 ID(쉼표구분)** 필드에 커스텀 스킬의 ID를 입력
- 예: `mySkill1, mySkill2`
- 스킬은 **스킬 DB** 탭에서 직접 만들거나 JSON으로 가져올 수 있습니다

### 등급별 스킬 수 가이드

| 등급 | 권장 스킬 수 | 비고 |
|:---:|:---:|------|
| E | 2~4 | 기본기 위주 |
| D | 3~5 | 특화 스킬 추가 |
| C | 4~6 | 성장형 스킬 포함 |
| B | 5~7 | 고급 스킬 배정 |
| A | 6~8 | 강력한 스킬 조합 |
| S | 7~10 | 최상위 스킬 |

## 9. 장비와 가방

- **가방**은 캐릭터 DB에서 설정하지 않습니다
- 아래쪽 **장비창**에서 가방 아이템을 장착하면 자동으로 적용됩니다
- 장비(무기/방어구/악세서리)도 장비창에서 관리합니다

---

## 10. 전체삭제

**"캐릭터 전체삭제"** 버튼을 누르면 모든 캐릭터가 삭제됩니다.
파티 슬롯과 팀 구성도 초기화됩니다.

---

## 11. 자주 하는 실수

| 실수 | 해결 |
|------|------|
| 스탯이 10 미만 | 모든 스탯 최소값은 **10** |
| 스탯합이 등급 기준 초과 | 등급별 스탯합 범위 확인 |
| 주스탯이 등급 상한 초과 | 상한표 확인 (E:25, D:40 등) |
| 스킬 ID 오타 | 스킬 DB에서 정확한 ID 복사 |
| 행과 포지션 불일치 | 탱커=front, 궁수/마법사=back |
| HP/MP/SP를 따로 안 넣고 싶다 | **0**으로 두면 스탯 자동 계산 |
