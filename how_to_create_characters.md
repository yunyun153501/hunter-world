# 👤 캐릭터 만드는 법 — Hunter World 플러그인 가이드

> 플러그인 UI의 **DB** 탭 → **캐릭터** 서브탭에서 캐릭터를 추가/수정/삭제할 수 있습니다.
> 이 문서는 캐릭터를 설계할 때 필요한 **모든 필드**와 **스탯 기준치**를 설명합니다.

---

## 1. 캐릭터 추가 방법

1. 플러그인 UI에서 **DB** 탭 클릭
2. **캐릭터** 서브탭 클릭
3. **"새 캐릭터"** 버튼 클릭
4. 아래 필드들을 채운 후 **"저장"** 클릭

---

## 2. 캐릭터 필드 설명

| 필드 | 설명 | 예시 |
|------|------|------|
| **ID** | 영어, 고유값 (자동 생성 가능) | `char_yuna` |
| **이름** | 캐릭터 이름 | `최유나` |
| **직업** | 캐릭터 직업 | `크루세이더` |
| **포지션** | 전투 역할 | `전열탱커` |
| **전열** | front / mid / back | `front` |
| **등급** | E / D / C / B / A / S | `E` |
| **HP** | 체력 (자동 계산 가능) | `290` |
| **MP** | 마나 (자동 계산 가능) | `130` |
| **SP** | 스태미나 (자동 계산 가능) | `240` |
| **ATK** | 공격력 (자동 계산 가능) | `12` |
| **물리방어** | PDEF | `7` |
| **마법방어** | MDEF | `4` |
| **STR** | 힘 | `12` |
| **CON** | 체질 | `19` |
| **INT** | 지능 | `3` |
| **AGI** | 민첩 | `14` |
| **SENSE** | 감각 | `15` |
| **피해 타입** | physical / magic | `physical` |
| **공격 스탯** | 주력 스탯 | `con` |
| **기본 위협값** | 탱커는 높게, 딜러는 낮게 | `0` |
| **스킬 ID** | 보유 스킬 (쉼표 구분) | `shieldBash, taunt` |
| **메모** | 설명 텍스트 | `전열 탱커` |

## 3. 직업 종류

| 직업 | 포지션 | 전열 | 특성 |
|------|--------|:---:|------|
| 크루세이더 | 전열탱커 | front | 탱킹 + 도발 |
| 무투가 | 근거리물리 | front | 근접 딜러 |
| 궁수 | 원거리물리 | mid/back | 원거리 물리 딜러 |
| 투척가 | 원거리물리 | mid | 투척 딜러 |
| 마법사 | 원거리마법 | back | 마법 딜러 |
| 클레릭 | 힐러 | back | 회복 + 서포트 |
| 정령술사 | 원거리마법 | mid | 소환 + 마법 |
| 연금술사 | 후열서포터 | back | 포션 + 지원 |
| 스트라이커 | 근거리물리 | front | 고화력 근접 |
| 수리가 | 비전투 | back | 장비 수리 |
| 대장장이 | 비전투 | — | 장비 제작 |
| 무직업 | 다양 | 다양 | 범용 |

## 4. 스탯 공식 (자동 계산 기준)

> ❗ 플러그인에서 직접 HP/MP/SP/ATK 등을 수동 입력해도 됩니다.
> 아래는 **스탯에 기반한 자동 계산 기준**입니다.

| 파생 스탯 | 공식 |
|-----------|------|
| **HP** | `100 + CON × 10 + STR × 3` |
| **MP** | `100 + INT × 10 + SENSE × 3` |
| **SP** | `100 + AGI × 10 + SENSE × 3` |
| **ATK** | `기본ATK(등급) + STR × 0.2 + AGI × 0.2 + INT × 0.3` |
| **PDEF** | `floor(CON × 0.4)` |
| **MDEF** | `floor((INT + SENSE) × 0.25)` |

### 등급별 기본 ATK

| 등급 | 기본 ATK |
|:---:|:---:|
| E | 5 |
| D | 15 |
| C | 25 |
| B | 45 |
| A | 70 |
| S | 100 |

## 5. ⚖️ 등급별 스탯 기준치

| 구분 | E | D | C | B | A | S |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| **최대 레벨** | 15 | 25 | 40 | 60 | 80 | 120 |
| **주스탯 상한** | 25 | 40 | 60 | 80 | 100 | 150 |
| **스탯합 기준** | ~70 | 70~90 | 90~120 | 120~160 | 160~200 | 200~280 |

### 스탯 배분 원칙

- **주스탯** (직업 핵심): 등급 상한의 50~80%
- **보조스탯**: 주스탯의 50~70%
- **약점스탯**: 낮게 유지 (3~10)
- **스탯 합계**: 등급별 기준 범위 내로

### 직업별 주스탯 가이드

| 직업 | 주스탯 | 보조스탯 |
|------|--------|----------|
| 탱커 (크루세이더) | CON | STR, SENSE |
| 근접 딜러 (무투가/스트라이커) | STR | AGI, CON |
| 원거리 물리 (궁수/투척가) | AGI | SENSE, STR |
| 마법사 | INT | SENSE, AGI |
| 힐러 (클레릭) | INT | SENSE, CON |
| 서포터 (연금술사) | INT, SENSE | CON |

## 6. 전열 배치

| 전열 | 코드 | 포지션 |
|------|------|--------|
| 전열 | `front` | 탱커, 근접 딜러 |
| 중열 | `mid` | 원거리 물리, 서포터 |
| 후열 | `back` | 마법사, 힐러 |

> 💡 포지션이 지정되면 전열이 자동으로 추론됩니다:
> - 탱커/크루세이더 → front
> - 궁수/투척가 → mid  
> - 마법사/힐러 → back

---

## 7. 스킬 배정

- **스킬 ID(쉼표구분)** 필드에 내장 스킬 또는 커스텀 스킬의 ID를 입력
- 예: `energyBolt, heal, haste`
- 내장 스킬 153종은 스킬 DB 탭에서 확인 가능
- 캐릭터 등급에 맞는 스킬을 배정하세요

### 등급별 스킬 수 가이드

| 등급 | 권장 스킬 수 | 비고 |
|:---:|:---:|------|
| E | 2~4 | 기본기 위주 |
| D | 3~5 | 특화 스킬 추가 |
| C | 4~6 | 성장형 스킬 포함 |
| B | 5~7 | 고급 스킬 배정 |
| A | 6~8 | 강력한 스킬 조합 |
| S | 7~10 | 최상위 스킬 |

## 9. 장비와 가방

- **가방**은 캐릭터 DB에서 설정하지 않습니다
- 아래쪽 **장비창**에서 가방 아이템을 장착하면 자동으로 적용됩니다
- 장비(무기/방어구/악세서리)도 장비창에서 관리합니다

---

## 10. 전체삭제

**"캐릭터 전체삭제"** 버튼을 누르면 모든 캐릭터가 삭제됩니다.  
파티 슬롯과 팀 구성도 초기화됩니다.

---

## 11. 자주 하는 실수

| 실수 | 해결 |
|------|------|
| 스탯합이 등급 기준 초과 | 등급별 스탯합 범위 확인 |
| 주스탯이 등급 상한 초과 | 상한표 확인 (E:25, D:40 등) |
| 스킬 ID 오타 | 스킬 DB에서 정확한 ID 복사 |
| 전열과 포지션 불일치 | 탱커=front, 마법사=back 등 |
| HP/MP/SP를 0으로 두기 | 스탯 공식 참고하여 계산 |

> 💡 **팁**: 기존 내장 캐릭터(buildSampleCharacters)를 참고하면 올바른 스탯 배분을 이해할 수 있습니다.  
> 내장 캐릭터는 코드의 `buildSampleCharacters()` 함수에 41명이 정의되어 있습니다.
